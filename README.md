# education-front-web

## Project setup
```
npm install
```

### 项目运行（开发环境）
```
npm run dev
```

### 项目运行（测试环境）
```
npm run test
```

### 项目运行（线上环境）
```
npm run prod
```

### 项目打包(开发环境)
```
npm run build:dev
```

### 项目打包(测试环境)
```
npm run build:test
```

### 项目打包(线上环境)
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
